Keywords
--------

bar_linewidth
^^^^^^^^^^^^^

default: **None**

line width for bars, if null the width of the curve line width is used

+-------------------------------------------------+-------------------------------------------------+
|attribute                                        |value                                            |
+=================================================+=================================================+
|unique                                           |True                                             |
+-------------------------------------------------+-------------------------------------------------+

box_linewidth
^^^^^^^^^^^^^

default: **0.5**

line width for boxes

+-------------------------------------------------+-------------------------------------------------+
|attribute                                        |value                                            |
+=================================================+=================================================+
|unique                                           |True                                             |
+-------------------------------------------------+-------------------------------------------------+

color
^^^^^

default: **None**

color of the bar or box. If not set, the color of the curve color is used

+-------------------------------------------------+-------------------------------------------------+
|attribute                                        |value                                            |
+=================================================+=================================================+
|optional                                         |True                                             |
+-------------------------------------------------+-------------------------------------------------+
|unique                                           |True                                             |
+-------------------------------------------------+-------------------------------------------------+

rescale
^^^^^^^

default: **False**

the plot is scaled using the values of the curves. If set to True, the values of the significance are also used to scale the Y axis, ensuring that the significance boxes or bars are visible

+-------------------------------------------------+-------------------------------------------------+
|attribute                                        |value                                            |
+=================================================+=================================================+
|unique                                           |True                                             |
+-------------------------------------------------+-------------------------------------------------+

type
^^^^

default: **bar**

none: no significance displayed, bar: error bars are used, box: boxes on the reference curve are displayed
