rank
^^^^

default: **100**

The rank determines at which point of drawing this particular comment should be drawn. Negatives values mean before any curves are plotted, positive values mean mean after plotting the curves. The comments are plotted in the order of the rank so the user has control over the order in which graphical comments are drawn.
