Keywords

alpha
^^^^^

default: **0.9**

this is a percentage of opacity for the alpha channel, 0 is transparent, 1 is full color

|unique                                           |True                                             |

automatic_edge
^^^^^^^^^^^^^^

default: **True**

when drawing bars, the edges of the bars are drawn in black by default. When the number of bars increases, the black of the edge may hide the actual bar color. In automatic edge mode, the black edge is removed, or changed to the value of edgewidth if it is defined when the number of bars exceeds the attribute edge_limit. If set to False, the edge is always drawn.

+-------------------------------------------------+-------------------------------------------------+
|attribute                                        |value                                            |
+=================================================+=================================================+
|unique                                           |True                                             |
+-------------------------------------------------+-------------------------------------------------+

bar_color
^^^^^^^^^

default: **wheat, olive, yellow, darkmagenta, tan, orange, salmon, rosybrown, palevioletred, cadetblue, seagreen, darkgoldenrod, brown, darkkhaki, lightslategrey, darkslateblue, navy, maroon, dodgerblue, firebrick, floralwhite, lemonchiffon, lightseagreen**

a list of colors which are affected in turn and in a cyclic way to each bar (part of a curve) in a single plot.

bar_width_factor
^^^^^^^^^^^^^^^^

default: **0.85**

when plotting bars, each bar is allocated a space (the length of its axis divided by the number of bars) to be plotted. This value is the percentage of that space. A value of 1 means that all bars will touch each others, less than 1 introduces gaps between them.
