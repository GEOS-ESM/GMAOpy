.. _plotlegend:

plotlegend
==========

The legend can be fully customized using those attributes. These attributes are native attributes from matplotlib, 

please see http://matplotlib.sourceforge.net/api/axes_api.html?highlight=legend#matplotlib.axes.Axes.legend
