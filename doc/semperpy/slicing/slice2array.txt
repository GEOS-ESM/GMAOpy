Methods

    .. method:: Slice2Array(generator = None):

    	:param generator: you can either supply an ArrayGenerator created with your own parameters or any object which has the same methods and signatures as the ArrayGenerator
    	:type generator: ArrayGenerator

        creates the destination array using the shape and the variable.

    .. method:: postProcess(dimensions,variable,shape,slices,metadata):
